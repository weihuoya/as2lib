This Package should only contain basic Exceptions an Exception here shouldn't be for a special class.

Packagemaster: Martin Heidegger

Contain (should Status):

ObjectAllreadyDefinedException - Exception if an Object should be created that is Allready defined.
ObjectNotDefinedException - Exception if an Object isn't defined that should be defined.
ObjectNotFoundException - Exception if an Object could not be found. (in association to searching!)
WrongArgumentException - Exception if an Argument for a Method wasn't correct.
