bindAdviceFactoryByAdviceClass

function bindAdviceFactoryByAdviceClass (
adviceType:Number, adviceClass:Function) : AdviceFactory
Creates a new SimpleAdviceFactory instance for the given
adviceClass, binds the created advice factory to the given
adviceType and returns the factory.

If there is already an advice factory bound to the given adviceType
the old binding gets overwritten.

Parameters:
adviceType :
the type to bind the adviceFactory to
adviceClass:
the class of the advice to create instances of if a advice
request for the given adviceType is made
Returns:
an advice factory configured for the given adviceClass
Throws:
org.as2lib.env.except.IllegalArgumentException if argument adviceType is null
or undefined
org.as2lib.env.except.IllegalArgumentException if argument adviceClass is null
or undefined

bindAdviceFactoryByAdviceFactory

function bindAdviceFactoryByAdviceFactory (
adviceType:Number, adviceFactory:AdviceFactory) : Void
Binds the given adviceFactory to the adviceType. If a request is
made with the given adviceType as type, the bound adviceFactory
will be used to create the advice that gets returned.

If there is already an advice factory bound to the given adviceType
the old binding gets overwritten.

If you want to remove a binding pass an adviceFactory of value
null or undefined

Parameters:
adviceType :
the type to bind the adviceFactory to
adviceFactory:
the advice factory to bind to the given adviceType
Throws:
org.as2lib.env.except.IllegalArgumentException if argument adviceType is null
or undefined

getAdviceByTypeAndPointcutAndCall

function getAdviceByTypeAndPointcutAndCall (
type:Number, pointcut:Pointcut, callback:Call) : Advice
Returns the advice corresponding to the given type. The returned advice
uses the passed-in pointcut and callback.

The callback is invoked if the execute method of the returned
advice is executed.

Commonly supported types are defined as constants in the
AbstractAdvice class.

Parameters:
type :
the type of the advice to return
pointcut:
the pointcut used by the returned advice
callback:
the callback that is executed if you invoke the execute
method on the returned advice
Returns:
the advice corresponding to the type and configured with the given
pointcut and callback
Throws:
org.as2lib.env.except.IllegalArgumentException if argument type is null or
undefined
Specified by:
DynamicAdviceFactory.getAdviceByTypeAndPointcutAndCall
