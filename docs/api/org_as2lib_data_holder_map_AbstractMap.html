Instance methods

populate

private function populate (
source) : Void
Populates the map with the content of the passed-in source.

Iterates over the passed-in source with the for..in loop and uses the variables'
names as key and their values as value. Variables that are hidden from for..in
loops will not be added to this map.

This method uses the put method to add the key-value pairs.

Parameters:
source:
an object that contains key-value pairs to populate this map with
