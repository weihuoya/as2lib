addAdviceByTypeAndStringAndMethod

private function addAdviceByTypeAndStringAndMethod (
type:Number, pointcut:String, method:Function) : Advice
Adds a new advice of the given type, for the given pointcut and
with the given method that is executed if a join point captured by the
pointcut is reached.

The advice is obtained by the getAdvice method of the advice factory
returned by org.as2lib.aop.AopConfig.getDynamicAdviceFactory method.

Parameters:
type :
the type of the advice
pointcut:
the pointcut represented by a string that determines which join
points are captured
method :
the method that contains the actions to be executed at specific
join points captured by the pointcut
Throws:
org.as2lib.env.except.IllegalArgumentException if argument method is null or
undefined
