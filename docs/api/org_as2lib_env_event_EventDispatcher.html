EventDispatchers are used to dispatch to EventListeners.
Implement the EventDispatcher for different ways to dispatch events. It is possible to add an EventDispatcher instance to a DelegatingEventBroadcaster.
See also:
org.as2lib.env.event.DelegatingEventBroadcaster#setEventDispatcher()

dispatchConsumable

public function dispatchConsumable (
event:EventInfo, listeners:Array) : Void
Dispatches the event to all EventListeners with the help of the EventInfo instance. Special about this operation is that it checks whether the EventInfo is consumed. If it is consumed the dispatching will abruptly stop.
Parameters
info :
the EventInfo to be passed to the event
listeners:
the EventListeners to dispatch to
