DateFormatter

DateFormatter formats a given date with a specified pattern.

Use the declared constants as placeholders for specific parts of the date-time.

All characters from 'A' to 'Z' and from 'a' to 'z' are reserved, although not
all of these characters are interpreted right now. If you want to include plain
text in the pattern put it into quotes (') to avoid interpretation. If you want
a quote in the formatted date-time, put two quotes directly after one another.
For example: "hh 'o''clock'".

Example:

var formatter:DateFormatter = new DateFormatter("dd.mm.yyyy HH:nn:ss S");
trace(formatter.format(new Date(2005, 2, 29, 18, 14, 3, 58)));

Output:

29.03.2005 18:14:03 58

formatDayAsNumber

privatefunction formatDayAsNumber (
day:Number, digitCount:Number) : String
Formats the passed-in day into a day as number string with the specified
digitCount.

A digitCount less or equal than one results in a day with one digit,
if the day is less or equal than nine. Otherwise the day is represented by a two
digit number. A digitCount greater or equal than two results in a day with
preceding 0s.

If the passed-in digitCount is null or undefined, 0
is used instead.

Parameters:
day :
the day of month to format to a number string
digitCount:
the number of digits
Returns:
the number representation of the day
Throws:
IllegalArgumentException if the passed-in day is less
than 1 or greater than 31 or null or undefined

formatHourInAmPm

privatefunction formatHourInAmPm (
hour:Number, digitCount:Number) : String
Formats the passed-in hour into a number string from range 1 to 12.

The resulting string contains only the specified digitCount if
possible. This means if the hour is 3 and the digitCount 1 the resulting
string contains one digit. But this is not possible with the hour 12. So in this
case the resulting string contains 2 digits. If digitCount is greater
than the actual number of digits, preceding 0s are added.

If the passed-in digitCount is null or undefined, 0
is used instead.

Parameters:
hour :
the hour to format
digitCount:
the number of favored digits
Returns:
the string representation of hour
Throws:
IllegalArgumentException if the passed-in hour is less
than 0 or greater than 23 or null or undefined
