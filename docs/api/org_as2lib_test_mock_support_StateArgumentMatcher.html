checkProperty

private function checkProperty (
target, propertyName:String, expectedPropertyValue) : Boolean
Checks whether the value of the property with name propertyName on the
passed-in target matches the expectedPropertyValue.

It is first checked whether a getter method with the passed-in
propertyName exists. If so it will be used to get the property value
that is then compared with the expectedPropertyValue. Otherwise the
propertyName will be used directly and its value will be compared with
the expectedPropertyValue.

Parameters:
target :
the target object that declares the property
propertyName :
the name of the property
expectedPropertyValue:
the expected value of the property
Returns:
true if the property has the expected value else false

matchArgument

function matchArgument (
argument) : Boolean
Checks whether the passed-in argument is in the expected state.

false will be returned if:

  • argument is null or undefined.
  • argument is not of the expected type.
  • argument is not in the expected state.
Parameters:
argument:
the argument to check whether it is in the expected state
Returns:
true if the argument is in the expected state else false
