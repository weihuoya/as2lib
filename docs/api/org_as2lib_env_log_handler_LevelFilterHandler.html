Constructor

LevelFilterHandler

function LevelFilterHandler (
handler:LogHandler, level:LogLevel)
Constructs a new LevelFilterHandler instance.

All log messages with a lower level than the passed-in level get
filtered out.

If level is not passed-in or is null or undefined all
levels will be allowed, this means it will be set to
org.as2lib.env.log.level.AbstractLogLevel.ALL.

Parameters:
handler :
the handler to forward not-filtered log messages to
(optional):
level the log level determining which log messages to filter
Throws:
org.as2lib.env.except.IllegalArgumentException if the passed-in handler is
null or undefined
