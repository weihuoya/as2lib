PropertyTestCaseByProperty

function PropertyTestCaseByProperty (
property:PropertyInfo, referenceScope, referenceName:String) : Void
Constructs a new PropertyTestCase by property.

If you want to profile a method, referenced from a different scope and with a different name you can specify thse with the last two arguments. Note that if specified the method declared on the class will not be profiled but its reference.

Parameters:
property :
the property to profile
referenceScope:
(optional) the scope of the property reference to profile
referenceName :
(optional) the name of the property reference to profile
Throws:
IllegalArgumentException if the passed-in property is null or undefined
