Constructor

SimpleStack

public function SimpleStack (
source:Array)
Constructs a new SimpleStack instance.

The stack steps through the passed-in source beginning at position zero and pushes all contained elements.

var stack:SimpleStack = new SimpleStack([1, 2, 3]);

 while (!stack.isEmpty()) {

      trace(stack.pop());

 }
The output is made in the following order: 3, 2, 1
Parameters
source:
(optional) an array that contains values to populate the new stack with
