ProxyFactory offers the functionality to create dynamic proxies, that means at run-time, of specific types, that are classes or interfaces.

Methods that get invoked on the returned proxy are forwared to an invocation handler that is responsible to respond appropriately to the method invocation.

createProxy

function createProxy (
type:Function, handler:InvocationHandler)
Creates a proxy for the given type and delegates method invocations to the InvocationHandler.invoke method.

Types are classes or interfaces.

The returned proxy can be casted to the passed-in type.

Parameters:
type :
the type to create a proxy for
handler:
the handler to call the invoke method on method invocations on the proxy
Returns:
a proxy for the given type
