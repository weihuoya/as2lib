Interface for listening to a Impulse.

Impulse allows two ways of listening to a certain impulse.
Executable with Impulse.connectExecutable allows in some cases only
bad readable usages.

With Impulse#addListener its possible to add a listener typed as
ImpulseListener. The method onImpulse will be called on each
impulse.
