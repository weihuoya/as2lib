TraceHandler

function TraceHandler (
Void)
Constructs a new TraceHandler instance.

You can use one and the same instance for multiple loggers. So think about using the handler returned by the static getInstance method. Using this instance prevents the instantiation of unnecessary trace handlers and and saves storage.
