Config is the basic configuration class of the as2lib framework.

It lets you configure global behavior of key features like the stirngification of classes and instances.

getObjectStringifier

public static function getObjectStringifier (
Void) : Stringifier
Returns the stringifier used to stringify objects.

If no stringifier is set the default stringifier gets returned. That is an instance of class ObjectStringifier.
