FlexApplication is the default access point for Flex applications.

Use this class to configure your application with Macromedia Flex. It inits your flex configuration in main.Flex and your configuration for all environments in main.Configuration.

You must simply init your application something like this:

<mx:Application xmlns:mx="http://www.macromedia.com/2003/mxml" initialize="initApp();">
  <mx:Script>
    <![CDATA[
      import org.as2lib.app.conf.FlexApplication;
      public function initApp(Void):Void {
        FlexApplication.init();
      }
    ]]>
  </mx:Script>
</mx:Application>
