ConsumableEventDistributorControl marks the EventDistributorControl
interface as consumable.

A distributor that implements this interface allows for consumable events.
This means that the event distribution is stopped as soon as an event is
consumed.

An event is consumed if a listener's event method returns true. If it
returns nothing, null or undefined or false the event
will further be distributed.
