JoinPoint represents an identifiable point in a program.
Although this points could also be try..catch blocks the join points offered by this framework are limited to members of classes or interfaces. Refer to http://www.simonwacker.com/blog/archives/000041.php for a description of join points in AOP terms.

getThis

function getThis (
Void)
Returns the logical this of the interception. That means if the join point is part of a call-pointcut the result will refer to the object where the call is made from. If the join point is part of an execution- pointcut the result will refer to the object the represented method resides in.
Returns:
the logical this of the join point depending on the wrapping pointcut

matches

function matches (
pattern:String) : Boolean
Checks if the join point matches the given pattern. The pattern can contain wildcards like '*' or '..'. A pattern could for example be 'org..BasicClass.*'. Refer to http://www.simonwacker.com/blog/archives/000053.php for an explanation of possible wildcards. Note the the '?' wildcard is not supported.
Parameters:
pattern:
the pattern to be matched against this join point
