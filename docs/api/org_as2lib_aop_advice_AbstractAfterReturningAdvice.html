executeJoinPoint

private function executeJoinPoint (
joinPoint:JoinPoint, args:Array)
Proceeds the passed-in joinPoint with the given args and returns
the result of this procession after invoking this advice's execute
method passing the given joinPoint and the result if the procession
returned successfully and did not result in an exception.
Parameters:
joinPoint:
the reached join point
args :
the arguments to use for the procession of the join point, these are
normally the ones that were originally passed-to the join point
Returns:
the result of the procession of the given joinPoint with the
given args
Throws:
if the procession of the joinPoint with the given args
results in an exception or if this advice's execute method threw an
exception
