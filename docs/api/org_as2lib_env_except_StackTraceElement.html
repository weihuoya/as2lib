StackTraceElement represents an element in the stack trace returned by the Throwable#getStackTrace method.

A stack trace element is simply a data holder.

It holds the thrower, the method that threw the throwable and the arguments passed to the method that threw the throwable.

This class is also responsible for stringifying itself. This functionality is used by the ThrowableStringifier. You can set your own stringifier to get a custom string representation of the stack trace element via the static #setStringifier method.
