addSpaceIndent

static function addSpaceIndent (
string:String, size:Number) : String
Adds a space indent to a existing String. This method is useful for different kind of ASCII Output writing. It generates a dynamic size of space indents in front of every line inside a string. Example:
var bigText = "My name is pretty important\n"
             +"because i am a interesting\n"
             +"small example for this\n"
             +"documentation.";
var result = StringUtil.addSpaceIndent(bigText, 3);
Result contains now:
My name is pretty important
 because i am a interesting
 small example for this
 documentation.
Parameters:
string:
String that contains lines that should get a space indent.
indent:
Size of the Indent (will get floor'ed)
Throws:
IllegalArgumentException if the size was smaller than 0
Returns:
String with a extended Indent.

contains

static function contains (
string:String, chars:String) : Boolean
Evaluates if a list of characters is contained in a string. This methods splits the searchstring(chars) and checks if any character is contained in another string(string). Example:
trace(StringUtil.contains("monkey", "kzj0")); // true
trace(StringUtil.contains("monkey", "yek")); // true
trace(StringUtil.contains("monkey", "a")); // false
Parameters:
string:
String that should be checked if it contains any of the searchstring.
chars :
Character String that contains a list of characters that should be checked if any character is contained.
Returns:
true if one of these characters are contained in string.
