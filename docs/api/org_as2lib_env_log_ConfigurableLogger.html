ConfigurableLogger declares methods needed to configure most loggers.

This is first the method to set the level setLevel and second the
methods to add, remove and access handlers addHandler,
removeHandler, removeAllHandlers and getAllHandlers.

Log handlers are responsible for making the actual output. There are a few
pre-defined handlers that can be used to log to different output devices. Take
a look at the org.as2lib.env.log.handler package for these.

Loggers are normally configured only once.
