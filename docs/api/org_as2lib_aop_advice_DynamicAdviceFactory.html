getAdviceByTypeAndPointcutAndCall

function getAdviceByTypeAndPointcutAndCall (
type:Number, pointcut:Pointcut, callback:Call) : Advice
Returns the advice corresponding to the passed type. The advice uses the passed pointcut and callback. The Callback will be executed when the Advice#execute(..):Void method is called.
Parameters:
type :
the type of the advice
pointcut:
the pointcut used by the returned advice
callback:
a Call instance to execute the execute method on when the advice is executed
Returns:
the advice corresponding to the type
