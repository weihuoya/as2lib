MethodInfo represents a method.

MethodInfo instances for specific methods can be obtained using the ClassInfo#getMethods or ClassInfo#getMethod methods.

That means you first have to get a class info for the class that declares or inherits the method. You can therefor use the ClassInfo#forObject, ClassInfo#forClass, ClassInfo#forInstance or ClassInfo#forName methods.

When you have obtained the method info you can use it to get information about the method.

trace("Method name: " + methodInfo.getName());

 trace("Declaring type: " + methodInfo.getDeclaringType().getFullName());

 trace("Is Static?: " + methodInfo.isStatic());
