matchArguments

function matchArguments (
expectedArguments:Array, actualArguments:Array) : Boolean
Compares the actual arguments only by type.

If a type is null or undefined the expected and actual argument will be compared directly with the strict equals operator.

False will be returned if:

  • The lengths of the expected and actual arguments differ.
  • The lengths of the expected types and the actual arguments differ.
  • Any actual argument is not of the expected type.
Parameters:
expectedArgumens:
the expected arguments
