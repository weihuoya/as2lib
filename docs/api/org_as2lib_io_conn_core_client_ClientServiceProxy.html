invokeByNameAndArgumentsAndCallback

public function invokeByNameAndArgumentsAndCallback (
methodName:String, args:Array, callback:MethodInvocationCallback) : MethodInvocationCallback
Invokes the passed-in method on the service, passing the arguments.

When the response arrives the appropriate callback method gets invoked.

If the passed-in callback is not null, the returned callback should be the same.

Parameters
methodName:
the name of the method to be invoked
args :
the arguments to pass to the method
