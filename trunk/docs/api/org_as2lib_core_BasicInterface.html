BasicInterface is the basic interface for each class in the as2lib framework.

It is recommended to always implement this interface in the classes of your own project but it is not a necessity.

You can use all functionality of the as2lib framework without implementing it.

In enables you to call the #theString method on instances that have been casted to interfaces.

The default implementation BasicClass offers an enhanced #toString method implementation that returns a better string representation than the default Object#toString method of Flash.
