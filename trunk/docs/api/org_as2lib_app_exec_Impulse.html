Impulse is a definition for events that gets executed periodically.

Periodical events could be frame executions, seconds, hours or dates. Impulse allows to seperate the certain kind of Impulse from the execution code.

The Impulse executes Executable.execute on each impulse to the connected executables. Example:

import org.as2lib.app.exec.Impulse;

var impulse:Impulse = new MySinusImpulse();
impulse.connectExecutable(new MySpecialCall());
impulse.connectExecutable(new MyMostImportantCall());
