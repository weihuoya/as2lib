Config is the basic configuration class of the as2lib framework.

It lets you configure global behavior of key features like the stirngification of classes and instances.

getObjectStringifier

static function getObjectStringifier (
Void) : Stringifier
Returns the stringifier used to stringify objects.

If no stringifier is set the default stringifier gets returned. That is an instance of class org.as2lib.core.ObjectStringifier.

Returns:
the currently used object stringifier

setObjectStringifier

static function setObjectStringifier (
stringifier:Stringifier) : Void
Sets a new stringifier used to stringify objects.

If you set a stringifier of value null or undefined getObjectStringifier returns the default stringifier.

Parameters:
stringifier:
the new object stringifier
