Flash is intended for configuration of applications compiled with Flash.

It allows you to define all Flash specific configurations similar to the
configuration in Configuration.

The current code uses a common configuration. If you have additional configuration
you have to overwrite (not extend!) this class in your application directory. The
only method that must be declared to be compatible is init.
