Default access point for a application in Flash.

Use this class for starting up your application with the Macromedia Flash compiler.
It will start your flash configuration in main.Flash and your configuration for all
environments in main.Configuration.

You have simply to add following code in the first frame action:

org.as2lib.app.conf.FlashApplication.init();

static function init (
) : Void
Executes the configuration for the flash environment in main.Flash and the
configuration for all environments in main.Configuration. These are the
main.Flash.init and main.Configuration.init methods.

This method takes any amount of arguments and forwards them to both methods
main.Flash.init and main.Configuration.init.
