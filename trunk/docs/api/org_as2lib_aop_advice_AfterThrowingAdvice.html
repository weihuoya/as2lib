execute

function execute (
joinPoint:JoinPoint, throwable) : Void
Executes the actions that shall take place after a specific join point threw
an exception.

If you use the proxy returned by the AbstractAfterThrowingAdvice#getProxy
method to overwrite the actual join point, this method is invoked after the
procession of the given joinPoint resulted in an exception.

Parameters:
joinPoint:
the join point this advice was woven into
throwable:
the throwable thrown by the given joinPoint
