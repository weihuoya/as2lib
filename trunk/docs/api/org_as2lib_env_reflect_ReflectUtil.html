ReflectUtil obtains simple information about members.

It is independent on any module of the As2lib. And thus does not include them
and does not dramatically increase the file size.

getVariableNames

static function getVariableNames (
instance) : Array
Returns an array that contains the names of the variables of the passed-in
instance as Strings.

The resulting array contains all variables' names even those hidden from
for..in loops. Excluded are only "__proto__", "prototype",
"__constructor__" and "constructor" and members that are of
type "function".

Note that it is not possible to get variables that have been declared in the
class but have not been initialized yet. These variables' names are thus not
contained in the resulting array.

This method will never return null. If the passed-in instance
has no variables an empty array will be returned.

Parameters:
instance:
the instance whose varaibles to return
Returns:
all initialized variables of the passed-in instance
