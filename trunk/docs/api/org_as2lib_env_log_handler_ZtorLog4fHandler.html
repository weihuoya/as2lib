ZtorLog4fHandler

function ZtorLog4fHandler (
messageStringifier:Stringifier)
Constructs a new ZtorLog4fHandler instance.

You can use one and the same instance for multiple loggers. So think about
using the handler returned by the static getInstance method. Using this
instance prevents the instantiation of unnecessary trace handlers and saves
storage.

Parameters:
messageStringifier:
(optional) the log message stringifier to use

privatefunction convertLevel (

Converts the As2lib LogLevel into a ZTOR Log4f level number.

The default level is Log4f.LOG4F. It is used if no match is found.

function write (

Writes the passed-in message using the Log4f.log method.

The string representation of the message to log is obtained via
the convertMessage method and passed as header to the Log4f.log
method.
