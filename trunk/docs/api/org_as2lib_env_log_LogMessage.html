LogMessage is a dumb model that contains all the information about the message to log.

These information are the the message to log, its level and the name of the logger that is responsible for logging the message.

The LogMessage#toString method uses the set stringifier to obtain its string representation.

If you want a different appearance of the log message you can use the static #setStringifier method to set your custom stringifier.

The LogMessageStringifier supports different presentation styles. It allows to switch the log level, the logger name and the time on and off.
