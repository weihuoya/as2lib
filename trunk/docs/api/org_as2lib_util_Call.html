Call enables another object to make call a method in another scope without having to know where.

This enables you to pass a call to another object and let the object execute the call. You use the execute method to do so.

forEach

function forEach (
object) : Void
Iterates over the passed-in object using the for..in loop and executes this call passing the found object, its name and the passed-in object.

Example:

class MyClass {

  private var a:String;
  private var b:String;
  private var c:String;

  public function MyClass() {
    a = "1";
    b = "2";
    c = "2";
  }

  public function traceObject(value, name:String, inObject):Void {
    trace(name + ": " + value);
  }

  public function listAll() {
    new Call(this, traceObject).forEach(this);
  }
}

Note that only members visible for for-in loops cause the execute method to become invoked.

Parameters:
object:
the object to iterate over
