MethodCallRangeByMinimumAndMaximumQuantity

privatefunction MethodCallRangeByMinimumAndMaximumQuantity (
minimum:Number, maximum:Number) : Void
Creates a new range with the passed-in minimum and maximum.
  • If minimum is null DEFAULT_MINIMUM will be used.
  • If maximum is null DEFAULT_MAXIMUM will be used.
  • If minimum is negative it will be made positive.
  • If maximum is negative it will be made positive.
  • If minimum is bigger than maximum the two values will be
    exchanged.
