TypedMap is used as a wrapper for org.as2lib.data.holder.Map instances that ensures that only values of a specific type get added to the wrapped map.

This class simply delegates all method invocations to the wrapped map. If the specific method is responsible for adding values it first checks if the values to add are of the expected type. If they are the method invocation gets forwarded, otherwise an org.as2lib.env.except.IllegalArgumentException gets thrown.
