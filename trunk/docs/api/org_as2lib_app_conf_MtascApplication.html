Default access point for a application with Mtasc.

Use this class for starting up your application with the MTASC compiler found at http://www.mtasc.org.
It will start your MTASC configuration in main.Mtasc and your configuration for all
environments in main.Configuration.

Simply use this class as main startup class in the mtasc preferences.

[MTASC directory]\mtasc.exe -cp "[your project path]" -cp "[as2lib project path]" org/as2lib/app/conf/MtascApplication.as

main

static function main (
container:MovieClip) : Void
Executes the configuration for the mtasc environment in main.Mtasc and the
configuration for all environments in main.Configuration. These are the
main.Mtasc.init and main.Configuration.init methods.

The Mtasc.init method is passed the container movie-clip, that
is by default _root if this class is used as main method for MTASC.

Parameters:
container:
the root movie-clip that is passed by MTASC to the main method
