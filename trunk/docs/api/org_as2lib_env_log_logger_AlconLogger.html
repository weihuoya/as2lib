AlconLogger delegates all messages to the net.hiddenresource.util.alcon.Alcon.trace method.

Using this class instead of the Alcon class in your application directly enables you to switch between almost every available Logging API without having to change the logging calls but just the configuration on startup.

Every global configuration must be done via the static methods on the Alcon class itself.
