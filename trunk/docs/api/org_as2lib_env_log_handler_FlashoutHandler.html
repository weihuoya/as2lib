FlashoutHandler

function FlashoutHandler (
Void)
Constructs a new FlashoutHandler instance.

You can use one and the same instance for multiple loggers. So think about using the handler returned by the static getInstance method. Using this instance prevents the instantiation of unnecessary flashout handlers and and saves storage.
