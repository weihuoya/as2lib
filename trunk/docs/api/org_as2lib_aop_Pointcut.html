Any class implementing this interface represents a pointcut.
A pointcut normally defines a pattern that can be matched against a join point to check whether the join point is being captured. Refer to http://www.simonwacker.com/blog/archives/000043.php for a explanation of what pointcuts are in AOP termonology.
