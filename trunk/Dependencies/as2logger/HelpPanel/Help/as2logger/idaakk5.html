PropertyHandler.handlePublisherProperties()

Handles appliable properties for publishers. Important: If a publisher or formatter is specified, make sure to import and reference the given class so it can be accessed by the Logging Framework

Usage

myPropertyHandler.handlePublisherProperties(name: String, publisher: String, formatter: String, level: String): Void

Parameters

name the Logger's name string, e.g. "a.b.c"

publisher the publisher class name, e.g. "com.domain.CustomPublisher"

formatter filter the formatter class name, e.g. "com.domain.CustomFormatter"
