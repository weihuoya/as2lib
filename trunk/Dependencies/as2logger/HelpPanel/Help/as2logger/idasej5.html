Level class

The Level class defines a set of standard logging levels that can be used to control logging output. The logging Level objects are ordered and are specified by ordered integers. Enabling logging at a given level also enables logging at all higher levels. Clients should normally use the predefined Level constants such as Level.SEVERE.

Author

Ralf Siegel

The Level class inherits from the Object class.

Field summary for the Level class

FieldDescription
Level.ALLALL indicates that all messages should be logged
Level.CONFIGFINEST indicates a highly detailed tracing message.

Level.FINERFINE is a message level providing tracing information.
Level.FINESTCONFIG is a message level for static configuration messages.
Level.INFOINFO is a message level for informational messages.
Level.OFFOFF is a special level that can be used to turn off logging.
Level.SEVERESEVERE is a message level indicating a serious failure.
Level.WARNINGWARNING is a message level indicating a potential problem.
