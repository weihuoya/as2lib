class TraceOutput

Object
  |
  +--logging.TraceOutput

Implemented Interfaces: IPublisher
class TraceOutput
extends Object

Standard implementation of the Logger's framework IPublisher interface. Plain text formats incoming logging messages and sends them (traces) to the output window.


Author:
Ralf Siegel

isLoggable ( logRecord: LogRecord ): Boolean
Check if this Publisher would actually log a given LogRecord. This method checks if the LogRecord has an appropriate Level and whether it satisfies any Filter. It also may make other Publisher specific checks that might prevent a publisher from logging the LogRecord.

public
publish ( logRecord: LogRecord ): Void
Publish a LogRecord. The logging request was made initially to a Logger object, which initialized the LogRecord and forwarded it here. The Handler is responsible for formatting the message.

public
setFilter ( filter: IFilter ): Void
Set a filter to control output on this Publisher. For each call of publish the Publisher will call this Filter (if it is non-null) to check if the LogRecord should be published or discarded.
