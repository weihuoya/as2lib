Class com.asual.enflash.managers.FocusManager

com.asual.enflash.EnFlashObject
   +--com.asual.enflash.managers.FocusManager

Description

FocusManager class controls the focus order and tabulation inside the EnFlash framework. It provides a transparent management for all the User Interface components.
