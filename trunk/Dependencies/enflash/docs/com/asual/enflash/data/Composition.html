Class com.asual.enflash.data.Composition

com.asual.enflash.EnFlashObject
   +--com.asual.enflash.data.Composition

Description

Composition class partly implements the Composite pattern and allows the creation of hierarchical structures. It's useful for describing application's data model and XML to Object conversion.
